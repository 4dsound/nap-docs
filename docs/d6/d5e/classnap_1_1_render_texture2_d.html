Description

Empty 2D GPU texture that can be declared as a resource in JSON or created at runtime. You can use this texture to store the result of a render pass by a nap::RenderTarget. When usage is 'Static' and 'Fill' is turned off the texture on the GPU is in an undefined state until being rendered to. This is ok when using the texture as a render target, before the texture is read somewhere else. All other usage modes initialize the texture to the specified clear color.
