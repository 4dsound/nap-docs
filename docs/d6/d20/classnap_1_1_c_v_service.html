Description

Manages the OpenCV library and is responsible for grabbing and forwarding frame data to capture components. Frames are only grabbed automatically when a frame is made available by a CVCaptureDevice and if a CVFrameCaptureComponent is interested in data from a capture device. If no capture component is interested in data from a capture device the frame will not be grabbed and decoded, saving potential compute cycles.
