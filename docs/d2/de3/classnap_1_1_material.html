Description

Resource that acts as the main interface to a shader. Controls how vertex buffers are bound to shader inputs. It also creates and holds a set of uniform struct instances, matching those exposed by the shader. If a uniform exposed by this material is updated, all the objects rendered using this material will use that same value, unless overridden by a nap::MaterialInstance.
