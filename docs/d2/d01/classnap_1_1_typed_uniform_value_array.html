Description

template<typename T>
class nap::TypedUniformValueArray< T >

List of uniform values, for example: float[3], vec3[10] etc. The list must be declared inside a uniform buffer object (block). myshader.frag example:

uniform UBO
{
uniform vec3 positions[10];
} ubo;
