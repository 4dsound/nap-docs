Description

Captures frames from a video streamed over the network. The captured video frame is stored on the GPU when hardware acceleration is available (OpenCL).

Add this device to a nap::CVCaptureDevice to capture frames from a video streamed over the network, in a background thread. Note that this object should only be added once to a nap::CVCaptureDevice!
