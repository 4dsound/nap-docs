Description

Interface for a specific type of OpenCV capture device (camera, video, network stream etc). Derive from this class to implement your own OpenCV adapter. The adapter itself does not schedule the frame capture operation, that is handled in the background by the nap::CVCaptureDevice. That device (a-synchronously) handles the frame grab and retrieve operations. The capture device also opens the adapter on startup and closes it when stopped. Note that every adapter should only be added once to a specific nap::CVCaptureDevice! Restart or remove the adapter if it throws an error during the capture operation, errors are often caused by a connection failure or end of stream.
