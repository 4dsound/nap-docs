Description

template<typename T>
class nap::SDOValue< T >

Single SDO (Service Data Object) value. Groups together useful SDO information such as the SDO index, subindex, description and value to get or set. This requires the slave to have a mailbox and to support the "CANopen over EtherCAT" (CoE) protocol.
